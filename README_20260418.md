### 리스트 (List) — `[]`

> **한 줄 요약**: 순서가 있고, 나중에 바꿀 수 있는 목록

```python
a = []                    # 빈 리스트 만들기
a = [a, 'SDG', True]     # 숫자, 문자, 불리언 등 다양한 타입을 한 리스트에 넣을 수 있음
print(a[1])              # 'SDG' → 0부터 세니까 1번 = 두 번째
print(a[0:2])            # [[], 'SDG'] → 0번부터 2번 직전(=1번)까지 잘라오기
```

**인덱싱이란?**  
- 리스트의 각 항목에는 번호(인덱스)가 붙어 있다
- `a[0]` = 첫 번째, `a[1]` = 두 번째 (0부터 시작한다!)
- `a[-1]` = 마지막, `a[-2]` = 뒤에서 두 번째

**슬라이싱이란?**  
- `a[시작:끝]` 형태로 일부를 잘라올 수 있다
- `a[0:2]` → 0번, 1번 (끝 번호는 포함 안 됨!)

### 딕셔너리 (Dictionary) — `{키: 값}`

> **한 줄 요약**: 이름표(키)를 붙여서 데이터를 저장하는 방식. 영어 사전처럼 단어(키)로 뜻(값)을 찾는 구조

```python
dic = {"홍길동": 90, "이순신": 80}

# 추가
dic["박하나"] = 89

# 수정
dic["홍길동"] = 100

# 삭제
del dic["이순신"]

# 출력 방법들
for k in dic.keys():       # 키만 순회
    print(k)

for v in dic.values():     # 값만 순회
    print(v)

for k, v in dic.items():   # 키와 값 동시에 순회 ← 가장 많이 씀!
    print(k, v)

# 특정 키가 있는지 확인
if "박하나" in dic:
    print("있어")
```

**리스트 vs 딕셔너리 차이**  
| | 리스트 | 딕셔너리 |
|---|---|---|
| 접근 방식 | 번호(인덱스)로 | 이름(키)으로 |
| 예시 | `a[0]` | `dic["홍길동"]` |
| 순서 | 있음 | Python 3.7+부터 삽입 순서 유지 |

```python
#         [0]   [1]   [2]    [3]    [4]    [5=-1]

# 은메달보다 동메달이 더 많은 나라 출력
for row in olympic:
    if row[2] < row[3]:       # 금메달 수 < 은메달 수
        print(row[1])         # 호주

total_gold = 0
for row in olympic:
    total_gold += row[2]
print(total_gold)             # 166

for row in olympic:
    gold = row[2] / row[-1] * 100   # row[-1] = 마지막 항목 = 합계
    if gold > 40:
        print(row[1], gold, '%')
# 독일, 독일, 북한
```

마이너스 인덱스는 뒤에서부터 센다.  
`row[-1]` = 마지막 항목, `row[-2]` = 뒤에서 두 번째

## 4. 단어 빈도수 분석 (딕셔너리 심화 활용)

텍스트에서 각 단어가 몇 번 나왔는지 세는 프로그램

```python
alibaba = "in a town of persia lived two brothers..."

# Step 1: 문장을 단어 단위로 쪼개기
word_list = alibaba.split()   # 공백(스페이스)을 기준으로 분리
# → ['in', 'a', 'town', 'of', ...]

# Step 2: 단어별로 등장 횟수를 딕셔너리에 저장
vocab = {}
for w in word_list:
    if w not in vocab:    # 처음 보는 단어면
        vocab[w] = 1      # 1로 시작
    else:                 # 이미 있는 단어면
        vocab[w] += 1     # 1 추가
```

**흐름 이해**  
- `split()` : 문장을 단어 리스트로 만들어줌
- `if w not in vocab` : 딕셔너리에 해당 단어가 없으면
- `vocab[w] = 1` : 새로 추가
- `vocab[w] += 1` : 이미 있으면 카운트 +1

---

### 빈도수 기준으로 정렬하기

```python
import operator

vocab_final = sorted(
    vocab.items(),              # 딕셔너리를 (단어, 횟수) 쌍의 리스트로 변환
    key=operator.itemgetter(1), # 각 쌍의 1번째(횟수)를 기준으로 정렬
    reverse=True                # 내림차순 (많은 것부터)
)
print(vocab_final)
# → [('he', 15), ('and', 13), ('to', 13), ...]
```

**`operator.itemgetter(1)` 이해하기**  
- `vocab.items()` 는 `('단어', 횟수)` 형태의 튜플 목록
- `itemgetter(1)` = "튜플의 1번째 값(횟수)을 기준 삼아라"
- `reverse=True` = 높은 숫자부터 정렬

## 6. 파일 읽기/쓰기

> 프로그램이 꺼져도 데이터를 저장해두고 싶을 때 파일을 사용한다

### 파일 쓰기 (`'w'` 모드)
```python
f = open("숫자.txt", 'w')   # 'w' = write(쓰기) 모드로 파일 열기. 파일이 없으면 새로 만들어줌
for i in range(1, 11):
    f.write(str(i * 100) + "\n")  # write()는 문자열만 받음 → str()로 변환 필수!
                                   # "\n" = 줄바꿈 문자
f.close()   # 파일을 다 쓰면 반드시 close() 해야 함! 안 하면 데이터 손실 위험
```

**주의:** `write()`는 숫자를 바로 넣을 수 없다. 반드시 `str()`로 문자열로 바꿔야 한다.

---

### 파일 읽기 (`'r'` 모드)

**방법 1: `readline()` - 한 줄씩 읽기**
```python
f = open("숫자.txt", 'r')   # 'r' = read(읽기) 모드
line = f.readline()          # 첫 번째 줄 하나만 읽기
print(line)                  # 100
f.close()
```

**방법 2: `readline()` + while 루프 - 전체 읽기**
```python
f = open("숫자.txt", 'r')
while True:
    line = f.readline()
    if not line:     # 더 읽을 줄이 없으면 (빈 문자열이면) 종료
        break
    print(line.strip())   # strip() : 앞뒤 공백과 줄바꿈(\n) 제거
f.close()
```

**방법 3: `readlines()` - 전체를 리스트로 읽기**
```python
f = open("숫자.txt", 'r')
lines = f.readlines()   # 모든 줄을 리스트로 한 번에 가져옴 → ['100\n', '200\n', ...]
for line in lines:
    print(line.strip())  # strip()으로 줄바꿈 제거하고 출력
f.close()
```

**세 가지 비교**
| 방법 | 특징 | 언제 쓰나 |
|------|------|---------|
| `readline()` | 한 줄씩 | 특정 줄만 필요할 때 |
| `readline()` + while | 전체, 한 줄씩 처리 | 줄마다 조건 처리할 때 |
| `readlines()` | 전체를 리스트로 | 전체를 한 번에 다루고 싶을 때 |

## 7. CSV 파일 다루기

> CSV = Comma Separated Values. 쉼표(,)로 구분된 텍스트 파일. 엑셀로 열 수 있음

### 기본 읽기
```python
import csv   # csv 라이브러리 불러오기

f = open(r"C:\Users\USER\python3ex\card.csv", encoding='utf-8')
# r"..." : 'raw string' → 백슬래시(\)를 그냥 문자로 읽어라
# encoding='utf-8' : 한글이 포함된 파일은 인코딩 지정 필수

data = csv.reader(f)   # csv 읽기 객체 생성
next(data)             # 첫 번째 줄(헤더/표 제목)을 건너뜀
data = list(data)      # 나머지 데이터를 리스트로 변환
f.close()
```

**`r"..."` 이란?**  
- 경로에 `\Users\` 같이 백슬래시가 있으면, Python이 `\U`를 특수문자로 오해할 수 있음
- `r`을 앞에 붙이면 "그냥 문자 그대로 읽어라"는 뜻

**`next(data)` 이란?**  
- CSV 파일의 첫 줄은 보통 [날짜, 상호명, 금액...] 같은 헤더(열 제목)
- 데이터 분석 시 헤더는 필요 없으므로 `next()`로 한 줄 건너뜀

```python
data = csv.reader(f)
next(data)
data = list(data)
f.close()

s_mon = [0, 0, 0]   # 10월, 11월, 12월 지출 합산용 리스트

for row in data:
    if row[-1] == '대표가맹':        # 승인상태가 '대표가맹'인 행만 처리
        mon = int(row[0].split('-')[1])   # '2019-10-12' → split('-') → ['2019','10','12'] → [1] = '10' → int = 10
        payment = int(row[-3])            # 금액 (이미 쉼표 없는 숫자라 가정)
        idx = mon - 10                    # 10월→0, 11월→1, 12월→2
        s_mon[idx] += payment

plt.rc('font', family='Malgun Gothic')
plt.title('10~12월 지출현황')
plt.bar(['10월', '11월', '12월'], s_mon, color='royalblue')
plt.show()

print(s_mon)   # [2093015, 4127744, 2023269]
```

**`split('-')` 이란?**  
- 문자열을 특정 구분자로 잘라서 리스트로 만들어줌
- `'2019-10-12'.split('-')` → `['2019', '10', '12']`
- `[1]` 로 월(10) 부분만 가져옴

**가게별 지출 딕셔너리 + TOP 10 그래프**

```python
import operator
import matplotlib.pyplot as plt

spending = {}
for row in data:
    if row[-1] == '대표가맹':

# 금액 기준 내림차순 정렬 후 상위 10개만 슬라이싱
top10 = sorted(spending.items(), key=operator.itemgetter(1), reverse=True)[:10]

top10_store, top10_amount = [], []
for t in top10:

plt.rc('font', family='Malgun Gothic')
plt.title('10~12월 지출 TOP 10')
plt.barh(top10_store, top10_amount, color='b')   # barh = 가로 막대 (이름이 길 때 유용)
```

## 8. 함수 (def)

> 자주 쓰는 코드를 묶어서 이름을 붙여놓는 것. 필요할 때 이름만 불러서 재사용 가능

### 기본 함수 (입력 없음, 출력 없음)
```python
def ggg():           # def = define(정의하다). 함수 이름은 소문자로
    print("함수ggg()")  # 들여쓰기 필수!

ggg()    # 함수 호출 1번
ggg()    # 함수 호출 2번 → 같은 코드를 반복할 때 유용
```

### 매개변수가 있는 함수 (입력 있음, 출력 없음)
```python
def ggg2(x, y):      # x, y = 매개변수(파라미터). 함수에 전달할 값의 자리
    print(x, y)

ggg2(100, True)      # 100 → x, True → y
ggg2("안녕하세요", 30)  # 다양한 타입을 넣을 수 있음
```

### 반환값이 있는 함수 (입력 있음, 출력 있음)
```python
def ggg3(x, y):
    z = x * y
    return z    # return : 계산 결과를 함수 밖으로 내보냄

val = ggg3(100, 200)   # 함수 실행 후 결과(20000)를 val에 저장
print(val)             # 20000
val = ggg3(3, 5)
print(val)             # 15
```

**함수를 왜 쓰나?**  
- 같은 코드를 여러 번 복사하지 않아도 됨
- 코드 길이가 줄고 수정이 쉬워짐
- 나중에 코드를 읽기도 쉬워짐

## 9. tkinter (GUI 창 만들기)

> tkinter = Python 기본 내장 GUI 라이브러리. 별도 설치 없이 창(윈도우)을 만들 수 있음

### 기본 창 만들기
```python
from tkinter import *   # tkinter의 모든 것을 가져오기

root = Tk()                          # 창(윈도우) 객체 생성
root.title('나의 프로그램')           # 창 제목
root.geometry('400x200+800+300')
# '가로x세로+왼쪽에서거리+위에서거리'
# 400x200 = 창 크기 (픽셀)
# +800+300 = 화면 왼쪽에서 800px, 위에서 300px 위치에 창을 띄움

root.resizable(False, True)   # (가로 크기 조절, 세로 크기 조절) → 가로는 고정, 세로만 조절 가능

root.mainloop()   # 이게 있어야 창이 꺼지지 않고 유지됨. 항상 마지막에 써야 함!
```

---

### 라벨(글자) 배치하기

**방법 1: `place()` - 좌표로 정확한 위치 지정**
```python
label1 = Label(root, text='안녕하세요!')
label1.place(x=200, y=100)   # 창 내부 좌표. 왼쪽 위가 (0,0)
```

**방법 2: `pack()` - 자동으로 순서대로 배치**
```python
label1 = Label(root, text='안녕하세요!', relief='groove')  # relief = 테두리 스타일
label1.pack()              # 위에서 아래로 차례대로 쌓임
label2 = Label(root, text='Hello!', relief='groove')
label2.pack(side='left')   # side: 'left', 'right', 'top', 'bottom' 방향 지정 가능
```

**방법 3: `grid()` - 표(행/열) 형태로 배치**
```python
label1 = Label(root, text='안녕!', relief='groove')
label1.grid(row=0, column=0)   # 0행 0열
label2 = Label(root, text='Hello!', relief='groove')
label2.grid(row=0, column=1)   # 0행 1열
label3 = Label(root, text='곤니찌와!', relief='groove')
label3.grid(row=1, column=0)   # 1행 0열
```

### 버튼과 이벤트 처리
```python
def sayhello():       # 버튼 클릭 시 실행될 함수
    txt.set('Hello!! ' * 3)   # StringVar에 값 설정

def cleartext():
    txt.set('')   # 빈 문자열로 초기화

from tkinter import *
root = Tk()
root.title('나의 프로그램')
root.geometry('400x200+800+300')
root.resizable(False, False)

txt = StringVar()   # 화면에 표시할 문자열 변수 (일반 str이 아닌 tkinter용 변수)

lbl1 = Label(root, textvariable=txt, width=40)   # textvariable로 StringVar 연결
lbl1.grid(row=0, column=0)

btn1 = Button(root, text='확인', width=10, command=sayhello)    # command = 클릭 시 실행할 함수
btn1.grid(row=0, column=1)

btn2 = Button(root, text='지우기', width=10, command=cleartext)
btn2.grid(row=1, column=1)

root.mainloop()
```

**`StringVar()`란?**  
- tkinter에서 라벨/버튼의 텍스트를 동적으로 바꾸려면 일반 `str` 대신 `StringVar()`를 써야 함
- `txt.set('값')` 으로 값 설정, `txt.get()` 으로 현재 값 읽기
- `textvariable=txt` 로 라벨에 연결하면 `txt.set()` 할 때 화면이 자동으로 바뀜
