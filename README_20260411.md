### 2. 문자열 (String)

```python
# 문자열 기본
s = "Hello, Python!"
s[0]        # 'H' (인덱싱)
s[0:5]      # 'Hello' (슬라이싱)
s[-1]       # '!' (음수 인덱스)
len(s)      # 길이

# 문자열은 불변(immutable) - 직접 수정 불가
# x[1] = 'y'  → 오류!
x = x[:1] + "y" + x[2:]  # 슬라이싱으로 우회

# 문자열 메서드
s.replace("Hello", "Hi")
s.upper()
s.lower()
s.count("l")
s.find("P")
```

**% 포맷팅:**
```python
print("이름: %s, 나이: %d" % ("홍길동", 20))
print("%10s" % "right")   # 오른쪽 정렬
print("%-10s" % "left")   # 왼쪽 정렬
```

### 3. 리스트 (List)

```python
lst = [1, 2, 3, 4, 5]

# 인덱싱 / 슬라이싱
lst[0]      # 1
lst[-1]     # 5
lst[1:3]    # [2, 3]

# 주요 메서드
lst.append(6)         # 맨 끝에 추가
lst.insert(1, 99)     # 인덱스 1에 삽입
lst.remove(99)        # 값으로 삭제
lst.pop()             # 마지막 요소 꺼내기
lst.pop(0)            # 인덱스로 꺼내기
lst.index(3)          # 값의 인덱스 반환
lst.count(2)          # 값의 개수
lst.sort()            # 오름차순 정렬
lst.sort(reverse=True) # 내림차순 정렬
lst.reverse()         # 순서 뒤집기

# 내장함수
max(lst), min(lst), sum(lst), len(lst)
```

### 4. 딕셔너리 (Dictionary)

```python
student = {"name": "홍길동", "score": 90, "grade": "A"}

# 접근 / 수정
student["name"]         # "홍길동"
student["score"] = 95   # 수정
student["email"] = "..."  # 추가
del student["grade"]    # 삭제

# 주요 메서드
student.keys()    # 키 목록
student.values()  # 값 목록
student.items()   # (키, 값) 쌍

# 멤버십 확인
"name" in student         # True
"name" in student.keys()  # True

# 반복
for key, value in student.items():
    print(key, ":", value)
```

**딕셔너리 정렬:**
```python
import operator
sorted_dict = sorted(student.items(), key=operator.itemgetter(1))
```

### 7. 실습 프로젝트

**할 일 목록 앱 (To-do List):**
```python
todo = {}
while True:
    cmd = input("1.추가 / 2.완료 / 3.목록 / 4.종료 >> ")
    if cmd == "1":
        task = input("할 일: ")
        todo[task] = False
    elif cmd == "2":
        task = input("완료한 일: ")
        todo[task] = True
    elif cmd == "3":
        for task, done in todo.items():
            print(task, "✓" if done else "...")
    elif cmd == "4":
        break
```

**학생 성적 관리:**

